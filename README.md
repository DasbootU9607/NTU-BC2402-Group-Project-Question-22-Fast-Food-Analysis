# NTU BC2402 Fast Food Analysis Project

## Project Description
Analysis of fast food nutritional data with 3D visualizations and healthiness scoring.

## Files
- `fastfood_analysis.ipynb` - Main analysis notebook
- `fastfood_healthiness_3d_plot.html` - 3D interactive visualization
- `fastfood_analysis_scatter_plot.html` - Scatter plot visualization
- `times_new_roman_fastfood_analysis.html` - Styled analysis report
- `healthiness_score_for_scatter_plot.csv` - Dataset with health scores
- `fastfood_healthiness_3d_plot.png` - 3D plot image
- `fastfood_healthiness_3d_plot.pdf` - 3D plot PDF

## Technologies
- Python
- Pandas
- Plotly
- Matplotlib
- Jupyter Notebook
